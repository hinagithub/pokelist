## 🚀 Getting Started

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/pokelist.git
cd pokelist

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm start
```

ブラウザで `http://localhost:3000` を開いてください。

## 🛠️ Tech Stack

- **React 18** + **TypeScript**
- **Material-UI** - UIコンポーネント
- **Axios** - HTTP通信
- **[PokéAPI](https://pokeapi.co/)** - ポケモンデータ
